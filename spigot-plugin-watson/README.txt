This folder is used in supporting Article 4 of the Developerworks series on Medical Minecraft.

The spigot-plugin-watson.zip file has the source code that can be extracted into $HOME/bluemix-minecraft folder.
The $HOME/bluemix-minecraft folder will already exist if you perform git clone https://github.com/scheema/bluemix-minecraft

The article that this folder refers to is available at
http://www.ibm.com/developerworks/cloud/library/cl-bluemix-minecraft-watson-trs-4/index.html
